## 说明

- 本项目为武汉大学第三学期《GIS可视化》可视化系统设计课程大作业，由仓库拥有者本人（钟源）设计开发。该系统为本人领导的团队开发的[另一团队项目](https://github.com/ashyseer/COVID-19-Service-Website)的疫情可视化部分，该项目正在持续更新中。

## 系统环境

- Windows 7或更高版本
- VS Code 1.47
- .Net Core 3.1
- Apache ECharts 4.8
- axios 0.19
